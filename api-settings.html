Firebase Cloud Messaging Setup Instructions

01. Create Firebase Project

  • Go to Firebase Console (console.firebase.google.com)
  • Create a new project or select existing one
  • Copy the Project ID from project settings

02. Generate Service Account

  • Go to Project Settings > Service Accounts
  • Click "Generate new private key"
  • Download the JSON file
  • Upload it to your server storage/app/firebase/ folder

03. Enable FCM API

  • Go to Google Cloud Console
  • Enable Firebase Cloud Messaging API
  • Make sure billing is enabled for your project

04. Configure Settings

  • Enter your Project ID above
  • Enter the path to your service account JSON file
Security Note: Keep your service account JSON file secure and never commit it to version control. Store it in a secure location on your server.
