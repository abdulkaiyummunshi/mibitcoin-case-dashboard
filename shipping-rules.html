Shipping rules

Configure rules to calculate shipping fee

Select country
If you want to set shipping fee based on location, you need to enable "Load countries, states, cities from plugin location" in Settings -> Checkout and import location data in Tools -> Import/Export Data.
If you want to set shipping fee based on zip code, you need to enable "Enable zipcode" in Settings -> Checkout and set zip code for store address.

When enabled, customers can purchase digital products without creating an account. They will receive download links via email.

When enabled, the store will only sell digital products. Physical product options will be hidden from product forms and checkout.

When enabled, you can add license codes to digital products. When disabled, license code functionality will be hidden from product forms.

When enabled, used license codes will be hidden from the license codes table in the product edit form. This helps keep the interface clean when you have many used codes.

When enabled, orders containing only digital products will be automatically marked as completed once payment is received.
